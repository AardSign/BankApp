# BankApp – Aplicação Bancária em Java

Projeto desenvolvido com o objetivo de praticar **lógica de programação**, **Programação Orientada a Objetos (POO)** e manipulação de dados em memória, utilizando **Java**.

A aplicação simula operações bancárias básicas e é executada diretamente no **terminal**, sem utilização de banco de dados ou interface gráfica.

##  Tecnologias Utilizadas
- Java
- Git

##  Características do Projeto
- Aplicação executada no terminal
- Dados mantidos em memória durante a execução
- Projeto sem persistência em banco de dados
- Estrutura focada em lógica e POO

##  Funcionalidades
- Criação de contas bancárias
- Depósitos
- Saques
- Consulta de saldo

##  Objetivo do Projeto
Projeto desenvolvido para fins educacionais, com foco no aprendizado de:
- Programação Orientada a Objetos
- Estruturação de classes
- Lógica de programação
